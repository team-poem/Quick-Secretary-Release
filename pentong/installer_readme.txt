뚝딱비서 — 한글 문서 AI 어시스턴트
========================================

한글(HWP) 문서와 엑셀 파일을 AI가 채팅으로 도와주는 데스크톱 앱입니다.

[필요 사항]
  - Windows 10/11 (64비트)
  - 한컴 한글 프로그램 (HWP 기능 사용 시)

[시작 방법]
  1. 뚝딱비서를 실행합니다
  2. 초기 설정 화면에서 '자동 설치' 버튼을 클릭합니다
  3. 'Claude 로그인하기' 버튼을 클릭하여 로그인합니다
  4. 설정 완료 후 바로 사용할 수 있습니다

[참고]
  - 한글 프로그램 없이도 엑셀 관련 기능은 사용 가능합니다
  - 파일을 채팅창에 드래그하면 바로 첨부됩니다
